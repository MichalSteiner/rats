Get started!
==============================
Welcome to `RATS` (Revealing Atmospheres with Transmission Spectroscopy) package. This package allows extraction of high-resolution transmission spectroscopy, in particular from fiber-fed spectrographs like ESPRESSO and HARPS (although modification for other packages is mostly trivial). 

.. important::

    This code is still under development and bugs might be present. In case you encounter some bug, please send the description to Michal.Steiner@unige.ch or create a new issue on GitHub. You can also use either of these to request a feature.

.. important::

    If you use this code, please cite Steiner et al. (2023). In future, more up-to-date reference will be available.
